Conditions d'utilisation de l'offre :
Dans ce qui suit, le terme "Google Ads" peut désigner Google Ads ou Google Ads Express, selon le cas.

  1. Cette offre est réservée aux clients justifiant d'une adresse de facturation dans le pays suivant : France. Un seul code promotionnel est autorisé par annonceur.
  2. Pour activer cette offre : Saisissez le code promotionnel dans votre compte avant le 31 décembre 2020. Pour bénéficier de l'offre, vous devez saisir le code dans un délai de 14 jours à compter de la première impression d'annonce diffusée via votre premier compte Google Ads.
  3. Pour bénéficier de l'avoir : une fois le code saisi, vos campagnes publicitaires doivent générer des frais d'au moins €25 (hors taxes) dans un délai de 30 jours. Notez qu'un paiement de €25 n'est pas suffisant. Le suivi des frais publicitaires commence une fois que vous avez saisi le code, jusqu'à ce qu'ils atteignent €25.
  4. Une fois les étapes 2 et 3 effectuées, l'avoir est généralement appliqué sur le récapitulatif de facturation de votre compte dans les cinq jours suivants.
  5. Les avoirs ne s'appliquent qu'aux frais publicitaires à venir, et non à ceux qui ont été générés avant la saisie du code.
  6. Une fois que vous aurez dépensé votre avoir, vous ne recevrez pas de notification. Les frais publicitaires supplémentaires vous seront facturés selon votre mode de paiement. Si vous ne souhaitez plus diffuser d'annonces, vous pouvez supprimer vos campagnes ou les mettre en veille à tout moment.
  7. Pour prétendre à un avoir promotionnel, vous devez recevoir des factures Google Ads et respecter les règles du programme.
